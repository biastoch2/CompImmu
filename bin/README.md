# README

## Usage

Prepare an input-file in the following format:
```
LLLLVAPAY
SQEAEFTGY
KLEGKIVQY
```

Run the following command: `python predictor.py <input-file> <output-file>`

The output will be written into the `<output-file>` in the following format:

Exmaple Output:
```
LLLLVAPAY	0
SQEAEFTGY	1
KLEGKIVQY	0
```
